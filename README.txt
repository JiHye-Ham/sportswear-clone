##Sportswear-clone
___
나이키 홈페이지를 참고하여, HTML, CSS, JavaScript만을 활용하여 클론 코딩한 프로젝트입니다.

##주요기능 
___
모달창, 이미지 슬라이드, 반응형 레이아웃(모바일, 태블릿, pc) - 검색 폼 UI 구현

##기술 스택 
___
HTML5, CSS3, JavaScript(ES6) -CSS grid, Flexbox -FontAwesome(아이콘)
